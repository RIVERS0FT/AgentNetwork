# ADR-018：剧本列表为无条件全量列表

> 状态：已被 ADR-029 和 SR-SCENE-05 替代

## 历史决定

本 ADR 曾规定剧本列表无查询参数、无搜索、无过滤、无分页，仅返回全部剧本名称。

## 替代理由

权威需求 `SR-SCENE-05` 要求查询已保存剧本并展示名称、状态及基本信息，至少支持按名称关键字和状态筛选。因此本 ADR 不再是目标设计合同，禁止以其历史内容阻止查询、状态展示或基本信息返回。

## 当前迁移要求

现有无条件名称列表可以作为迁移期实现，但必须明确标记为“部分实现”，并按 [ADR-029](ADR-029-SR需求基线与追踪规则.md) 迁移接口、模型、前端和测试。
