# ADR-016：剧本列表与详情模型分离

> 状态：已接受；列表字段合同由 ADR-029 和 SR-SCENE-05 修订

## 继续有效的决定

- 剧本列表使用轻量列表项，单个剧本详情使用 `SceneSummary`；
- 预览能力可以复用只读 `SceneSummary`，不要求新增 `ScenePreview` 类型；
- 查看详情或预览不得创建仿真实例、分配容器或修改运行状态；
- 列表不得返回完整 Agent、Skill、Tool 或拓扑详情。

## 已被修订的内容

旧合同中“列表只包含 `scene_key` 和 `title`”“不显示状态和基本信息”的限制已被 [ADR-029](ADR-029-SR需求基线与追踪规则.md) 与 `SR-SCENE-05` 替代。目标列表必须支持名称关键字和状态查询，并返回标识、名称、可用状态及基本信息。

当前生产模型仍偏向旧合同，属于待迁移实现，不得反向削弱 SR。
