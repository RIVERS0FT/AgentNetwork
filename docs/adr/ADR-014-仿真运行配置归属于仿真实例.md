# ADR-014：仿真运行配置归属于仿真实例

> 状态：已接受；字段合同由 ADR-019 修订

`SimulationRuntimeConfig` 归属于 `SimulationRun`，不属于 `SceneDefinition`、`SceneResource`、`SceneSummary` 或 `SceneListItem`。

运行配置按 `simulation_id` 保存，事件调度和终止逻辑只读取当前实例配置；读取剧本不得修改运行状态、事件队列或终止配置。不得因当前只允许一个仿真，就把运行配置设计成无 `simulation_id` 所有权的全局合同。

持续时间约束和事件驱动字段以 [ADR-019](ADR-019-持续时间约束的事件驱动仿真.md) 为准。
